前言

今天发现了一个好工具，mdbook,通过它可以更好地整理我们的学习笔记，下面，我来介绍一下该工具的基本使用方法

首先下载Rust的安装器：
官网：https://www.rust-lang.org/zh-CN/tools/install

下载完成后，双击exe，会弹出一个命令窗口，输入 1 ，之后等待

再在命令窗口中输入 rustc –version,如图示，即成功安装

之后 再下载 mdbook及相关构建工具（cargo为rust的包管理工具，如同python里的pip）
cargo install mdbook
cargo install mdbook-linkcheck
cargo install mdbook-toc
cargo install mdbook-open-on-gh

待下载完上述工具后，我们就可以开始构建mdbook项目了
